## 接口返回值
| 属性            | 说明                   |
|-----------------|------------------------|
| _id             | id                     |
| all             | 所有受感染的人数       |
| area            | 地区                   |
| array.all       | 截至time所有的感染人数 |
| array.desc      | 描述                   |
| array.num       | time当日的感染人数     |
| array.post_time | 来源文章发布的时间     |
| array.source    | 来源                   |
| array.time      | time                   |
